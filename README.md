Pyversion
=========

Python installer for fish shell.

Install
-------

For [fisher](https://github.com/jorgebucaran/fisher) user:

```bash
$ fisher add maralla/pyversion
```

Commands
--------

List all available versions:

```bash
$ pyversion versions
```

Install a Python version:

```bash
$ pyversion install 3.9.1
```
